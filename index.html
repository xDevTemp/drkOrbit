User Manual

This section has the goal to help a new user around.
Requirements:
- Internet Connection (doesn't need to be the fastest);
- DrkOrbit program version 0.0.1 alpha
- Darkorbit account level 18+* (premium version doesn't have a level cap)
* Read the Q&A section for better understanding on how it works

1. Login
1.1 When prompted with the login page it is necessary to enter the account credentials.
1.2 A message will prompt saying "Logged in", in case this doesn't appear check Known Bugs.
1.3 A new page will appear with the account information and all the items.
2. Verification
2.1 It is necessary to verify the email.
2.2 This process can take up to 2 minutes depending on the server status.
3. Item selection
3.1 Up to 8 items can be selected more than 8 isn't permitted, not including premium.

Known Bugs

1. After the user fills the login information the message "Logged in" doesn't appear and the application freezes. FIX: Minimize all windows and click on the opened application and it shall work.
2. Random freezes in the application.
3. Email not verified with correct information OR application Stuck in the Login page. FIX: Run the hotfix.exe as administator and try again.

Q&A

Q: How do we make money?
A: As of right now, we do not get any money with this application. Fist we want to build a community and then profit with that. Just like DISCORD does, after a community is build there is a subscription that gives bonus, in our case when we have a big enough community we will launch a premium version that has much more benefits. Long story short, we will only make money when we have a significant number of users.

Q: Why is it needed a email and respective information?
A: This is a really hard question to make a clear answer. The application works in a way that it "fakes" payments to darkorbit. This exploit makes it possible to do what we please given that the email is accessible. If the application used the user account instead of the email the account would be banned and severe consequences could happen. Since Bigpoint doesn't control any email provider, they do not have any idea that something is wrong. The accepted providers are Outlook and mail.com. Other providers (gmail) still doesn't let the application verify anything and there are plenty of providers and we can't make updates for all of them, for now only Outlook or mail.com are 99% sucessfull (mail.com has the best results). If you don't have any of those we recommend creating a new email and use that email. It is NOT recommended to use your personal email.

Q: Why is there a level cap?
A: This is something we decided and agreed inside our team. We did not want to let everyone use this or there would be no point on creating the application because it would be patched. We wanted to be able to support players that actually play this game and can't buy items in the game. The other reason for this is that we need features for our Premium application (in development), in this application we will not cap any level.

Q: How many servers does this application connects?
A: As of now we have 3 servers, 2 in Europe and 1 in America. The application will automatically connect to the fastest server. Note that the servers are cheap and can't handle too much traffic.
